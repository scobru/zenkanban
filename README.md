# 📋 ZenKanban — Decentralized E2EE Drag & Drop Project Board

ZenKanban is a lightweight, responsive, zero-knowledge **Trello-style Drag & Drop Kanban Board** running serverless inside the browser. It secures all your cards, tasks, and project workflows locally using WebCrypto API before syncing them across devices via `@akaoio/zen`.

Powered by **ZEN**—a zero-config decentralized peer-to-peer graph database.

## 🚀 Features

- 📋 **Interactive Drag & Drop**: Move cards seamlessly across columns (*Backlog, To Do, In Progress, Done*).
- 🏷️ **Rich Card Details**: Priorities (*Urgent, High, Medium, Low*), tags, markdown descriptions, checklists, and due dates.
- 🔐 **End-to-End Encrypted**: All tasks encrypted in-browser via master passphrase.
- ⚡ **Real-Time P2P Sync**: Live updates across all connected devices.
- 🎨 **Themeable & Fast**: Light & Dark themes, Outfit & IBM Plex Mono typography.
